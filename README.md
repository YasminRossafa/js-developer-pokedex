# 📖 Pokédex — JavaScript Learning Project

Este projeto foi desenvolvido durante o aprendizado de **JavaScript**, com foco nos seguintes conceitos:

- Consumo de APIs REST com `fetch`
- Manipulação do DOM
- Programação assíncrona com `Promises` e `async/await`
- Classes e orientação a objetos em JS
- CSS Grid e responsividade
- Navegação entre páginas com parâmetros na URL (`URLSearchParams`)

---

## 🚀 Funcionalidades

- Listagem dos **151 Pokémons originais** (Geração I) com tipos e cores
- Botão **Load More** para carregar mais Pokémons gradualmente
- **Página de detalhes** de cada Pokémon com estatísticas (HP, Ataque, Defesa, etc.)
- Navegação entre Pokémons direto na página de detalhes
- Design responsivo (mobile-first)

---

## 🛠️ Tecnologias

- HTML5 + CSS3
- JavaScript (ES6+)
- [PokéAPI](https://pokeapi.co/) — API pública e gratuita

---

## 📚 O que aprendi

Este projeto foi construído seguindo um curso de **JavaScript do zero ao avançado**, cobrindo:

1. **Fetch API** — como fazer requisições HTTP e encadear `.then()` para tratar respostas
2. **Promise.all** — como paralelizar múltiplas requisições e esperar todas terminarem
3. **Classes JS** — modelar entidades do mundo real com `class` e propriedades
4. **DOM manipulation** — criar e inserir elementos HTML dinamicamente
5. **URLSearchParams** — passar dados entre páginas sem back-end
6. **CSS Grid** — criar layouts responsivos com múltiplas colunas

---

## ▶️ Como usar

Basta abrir o `index.html` em um servidor local. Você pode usar:

```bash
# Com Node.js (npx)
npx serve .

# Com Python
python -m http.server 8000

# Com VS Code: instale a extensão "Live Server" e clique em "Go Live"
```
